Will radio and checkboxes be rendered correctly?

JsPDF doesn't have Radio Button from HTML capabilities it seems. There is a list of supported elements, and radio buttons aren't part of it. Link to stackoverflow
Radios with values 0, 1, 2. Checked works.
Radios with values 1, 2. Checked does not work.
Radios with values 1, 0. Checked does not work on any value.
Radios with values one, two. Checked does not work.
Radios with values 0, 2, 1. Checked will only work on values 0 and 1.
